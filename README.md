# Site da Agromiteon - Backend
Nesse diretório, reside o backend do website que desenvolvi para atender os interesses da empresa Agromiteon.

## Tecnologias Utilizadas
- Node.js
- Express.js
- MongoDb

Utilizei do Node.js com o Express para desenvolver a função de assinar newsletter, disponivel no final do site. O backend recebe os dados do usuario que o frontend manda e faz um post no banco de dados MongoDb, onde ficam armazenados todos os inscritos.
